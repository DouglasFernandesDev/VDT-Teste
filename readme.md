* Google Cloud Platform
* Maria BD
* DBeaver

## Techs, Stacks & Skills
* NodeJS
* Express

### Comandos
Para rodar o Servidor tenha o NPM ou o YARN instalados na máquina.
* `$ npm start` ou `$ yarn start`
* `$ npm run dev` ou `$yarn dev`
